PROMPTS UTILIZADOS PARA CRIAR O PODCAST :

# Nome do podcast

Você é um roteirista de podcast, e vamos criar um podcast de tecnologia, focado em front end e eu gostaria de uma ajuda sua para criar 5 sugestões
de nomes criativos para um podcast de front end feito por nerds, e que tenha algum trocadilho nerd no nome

O podcast vai falar sobre dicas e novidades sobre o mundo do front end e o que está acontecendo no mercado

{REGRAS}

- O nome deve ser enxuto, um nome e um subtítulo
- O nome tenha algum trocadilho nerd com nomes de franquias conhecidas como harry potter, star wars ou senhor dos anéis
- O nome deve conter alguma palavra forte que remeta a front end


# Episódio de Podcast

Você é um roteirista de podcast, e vamos criar um  roteiro de um podcast de tecnologia, focado em frontend cujo o nome é "O Enigma do Layout - A magia por trás da tela que você vê" e tem foco em frontend,  com o público alvo de iniciantes em frontend

o formato do roteiro deve ser
[INTRODUÇÃO]
[CURIOSIDADE 1]
[CURIOSIDADE 2]
[FINALIZAÇÃO]

{REGRAS}

- no bloco [INTRODUÇÃO] substitua por uma introdução iguais as introduções dos vídeos do canal 'ei nerd', como se fossem escritos pelo Peter Jordan
- no bloco [CURIOSIDADE 1] substitua por uma curiosidade de CSS
- no bloco [CURIOSIDADE 2] sobre uma ferramenta para front ends
- no bloco [FINALIZAÇÃO] substitua por uma despedida cool com o final 'Eu sou Gabi e esse foi o CodeQuest dessa semana'
- use termos de fácil explicação
- O podcast vai ser apresentado somente por uma pessoa, chamada Gabi
- O podcast deve ser curto
- - {REGRAS NEGATIVAS}
- Não use muitos termos técnicos
- Não ultrapasse 5 minutos de duração
